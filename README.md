## 🇨🇳 简体中文

U校园AI自动刷时长助手 —— 智能高效课程时长管理

### 简介

U校园AI自动刷时长助手是为U校园(新视野大学英语)平台开发的用户脚本工具。通过自动识别目录、智能时长分配和全流程自动化，帮助学生高效完成课时考核,节省时间专注于真正的学习内容。

### 主要功能

- ✅ **自动遍历目录** - 智能识别Unit、Section、Micro各级目录并自动切换
- ✅ **自动点击Tab/Task** - 支持多层级Tab与Task全自动点击
- ✅ **时长智能分配** - 总课时自动均分到每个学习子项，精确到每步每页
- ⏱️ **倒计时显示** - 每个页面显示本步倒计时，进度一目了然
- ⏸️ **随时暂停继续** - 随时暂停/继续，中途可修改配置并立即生效
- 🔄 **自动弹窗处理** - 自动识别并关闭「我知道了」、「确定」等弹窗
- 📝 **操作日志显示** - 全过程实时日志，出错可查
- 📱 **悬浮球启动** - 美观浮动球入口，UI简洁现代

#### 2. 安装脚本

#### 3. 使用步骤
1. 打开U校园课程学习平台页面 (https://ucontent.unipus.cn/*)
2. 点击页面右下角的紫色悬浮球 🎓
3. 在控制面板中选择起始目录及设置总刷课时长(分钟)

6. 日志区域会实时显示所有步骤和进度

- 🔒 **안전하고 신뢰할 수 있음** - U캠퍼스 페이지에서만 실행, 개인 정보 수집 안 함
- 🎨 **사용자 친화적 인터페이스** - 간결하고 아름다운 조작 패널
- ⚡ **스마트 대기** - 네트워크 지연에 자동 적응, 높은 성공률 보장
